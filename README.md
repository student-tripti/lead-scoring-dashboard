# 🚀 Lead Scoring Dashboard

## 📌 Overview
This project is an interactive web application built using Streamlit to score and rank leads based on multiple factors like engagement, website activity, company size, and industry score.

---

## 🎯 Features
- 📂 Upload CSV file
- 📊 Automatic lead scoring
- 🏷️ Categorization (High / Medium / Low)
- 🌍 Region-based filtering
- 📈 Score visualization (charts)
- 🏆 Best lead highlight
- 📥 Download filtered results

---

## 🧠 Scoring Logic
- Engagement → 40%
- Website Activity → 30%
- Company Size → 20%
- Industry Score → 10%

---

## ⚙️ Tech Stack
- Python
- Streamlit
- Pandas

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
